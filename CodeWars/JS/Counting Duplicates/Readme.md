# Ejercicio Counting Duplicates

## Solución

La función cuenta cuántos caracteres duplicados (insensibles a mayúsculas/minúsculas) existen en la cadena de entrada. Utiliza dos bucles anidados para comparar cada carácter con los siguientes y una lista para rastrear los caracteres ya contados, asegurando que cada duplicado se cuente solo una vez.
